# Percepti

Percepti is a native C++ desktop application that helps visually impaired users understand visual information displayed on their screen.

Unlike traditional screen readers that primarily read text, Percepti focuses on interpreting visual content such as charts, diagrams, graphs, presentation slides, and other graphical elements. The application captures a selected region of the screen, extracts relevant information using OCR and computer vision, and generates contextual explanations through a multimodal AI model.

This project was created to explore the intersection of accessibility, computer vision, and systems programming while developing a performant native desktop application in C++.

---

## Motivation

Modern accessibility tools provide excellent support for reading text but often struggle to explain the meaning behind visual information. This creates challenges for visually impaired students and professionals working with data visualizations, technical diagrams, and presentation materials.

Percepti aims to bridge that gap by transforming visual content into structured, meaningful explanations that improve comprehension and independence.
